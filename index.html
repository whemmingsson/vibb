  • Right click a gate to copy it
  • Hover on gate and press D to delete it
  • Click, hold and drag to move a gate
  • Hover on a gate and press I to create an input pin
  • Hover on a gate and press O to create an output pin
  • Click on the left vertical bar to create a button
  • Click on the right vertical bar to create an output
  • Click, hold and drag on an output pin / button to start creating a wire; drop it on an input pin / output
  • Right click on either a button, input pin, output pin or output to delete it
  • Right click on a wire to delete it
  • Click on a wire to create an anchor (it can be dragged in the same way as a gate)
  • Click on a button or an input pin (without ingoing wires) to turn it on.
